# 后端技术面试复习手册

## 简介

这是一份系统化的后端技术面试复习资料，涵盖分布式系统、并发编程、消息队列、数据库等核心技术领域的深度知识点。

## 目标读者

- 准备后端技术面试的工程师
- 希望深入理解并发编程和分布式系统的开发者
- 需要系统复习核心技术知识的从业人员

## 内容特点

- **深度优先**：每个知识点都深入到原理层面
- **面试导向**：围绕常见面试问题组织内容
- **实战相关**：结合实际应用场景讲解
- **持续更新**：保持内容的时效性和完整性

## 使用建议

1. 按章节系统学习，理解核心原理
2. 重点关注"面试要点"部分
3. 结合实际项目经验思考应用场景
4. 定期回顾，加深记忆
